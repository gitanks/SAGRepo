Use the Services screen to view all services currently deployed to Mediator.

Parameter
Description

Click on a particular service’s VSD link to view its virtual service definition (VSD).
WSDL
Click on a particular service’s WSDL link to view its WSDL.

Command
Action

Save
Click to save the modified configuration parameters if any.
