Runtime Alias Details : View Runtime Alias
View Runtime Alias
To promote a virtual service you can define runtime aliases to be used in Development stage, Test stage, and Production stage. These aliases are deployed to the runtime gateways along with the virtual services, and are referenced by the virtual services at runtime.
Use the Runtime Alias screen to display the list of all aliases that are deployed to Mediator.
Select and expand a runtime alias to view more details about it. The various details visible are:
Parameter
Description
Runtime Alias Type
The type of runtime alias. For example, Simple Alias, Secure Alias, Endpoint Alias, and so on.
Value
The value specified for a runtime alias. For example, http://myhost:5555
Authentication
The type of authentication for the runtime alias. For example, KerberosOutboundAuthentication.
Service Name
The name of the service for the runtime alias.
Key
The client's private key to be used for performing SSL client authentication.
The various operations performed in this screen are:
Command
Action
Expand All
Click to expand a runtime alias to view more details about it.
Collapse All
Click to collapse the runtime alias details.
