webMethods Mediator
Mediator is a service mediation and policy enforcement application for Web services. The Web services can be SOAP-based, REST-based, plain XML Web, or OData services. Mediator can also be used with Web services that have been externalized as Application Programming Interfaces (APIs).
Mediator is designed for use with Software AG Service-Oriented Architecture (SOA) products. The Mediator application is delivered as a package called WmMediator which runs on Integration Server. It provides an infrastructure for the run-time enforcement of service policies that are defined and managed from Software AG's UDDI registry/repository or CentraSite.
Mediator serves two primary functions:
To serve as an intermediary between service consumers and service providers (mediation).
To serve as a policy enforcement point (PEP) that enforces policies defined for a Web service.
