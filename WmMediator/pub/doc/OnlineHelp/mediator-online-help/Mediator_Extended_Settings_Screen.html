Mediator Configuration Settings : Extended Settings
Extended Settings
Use the Extended Settings screen to view and/or edit the parameters specified in the Mediator pg-config.properties file available in the <Integration Server_directory>\instances\instance_name\packages\WmMediator\config\resources folder.
You can directly edit the parameters from this screen instead of navigating to the location where the pg-config.properties file is placed. These parameters affect the operation of your server. You must not modify these settings unless requested to do so by Software AG Global Support.
Click Edit Extended Settings to edit the configuration parameters.
The various parameters that can be configured are:
Extended Settings
To edit the configuration parameters. For descriptions of the configuration parameters, see "Advanced Settings" in the Administering webMethods Mediator guide.
Note:  
Use care when editing these parameter settings. The editor makes no effort to ensure that your changes are valid. Whatever you type in this text box is saved in the pg-config.properties file.
