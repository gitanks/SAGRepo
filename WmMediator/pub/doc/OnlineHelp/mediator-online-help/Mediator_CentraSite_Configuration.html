Administering Mediator : CentraSite Configuration
CentraSite Configuration
The CentraSite Configuration screen is used to enable Mediator to report performance metrics and events to CentraSite.
You have to configure Mediator to report performance metrics and events to CentraSite.
Click Edit to edit the configuration parameters.
The various parameters that can be configured are:
Parameter
Description
Protocol
HTTP (the default) or HTTPS.
Host Name
The host name or IP address of the machine on which CentraSite is running.
Target Name
The name of the Mediator instance configured as a target in CentraSite. This name must match the target name defined in CentraSite.
UDDI Port
The port used for UDDI access to CentraSite. The default is 53307.
User Name
The CentraSite user name that Mediator must use to access CentraSite.
If you are using the Operating System auth mechanism, use the following format for the user name: CS-Host-Name\CS-user-name
The CentraSite user name and password that Mediator must use to access CentraSite.
If you are using the Operating System auth mechanism, use the following format for the user name: CS-Host-Name\CS-user-name
Password
The CentraSite password that Mediator must use to access CentraSite.
Retype Password
The same password you entered in the Password field.
Report Performance Data
Specify whether Mediator must collect and report performance metrics to CentraSite. The default is true.
Note:  
In order to save the configuration parameters, you must select Report Performance Data before you click Save. You can then clear it and click Save if you do not want to report the performance data.
Publish Interval (minutes)
Specify how often (in minutes) Mediator must report performance metrics. Enter a value from 1 through 60. The default is 60.
Note:  
You can only enter a value for this parameter if Report Performance Data is selected.
The various operations performed in this screen are:
Command
Action
Return
Click to return to the previous screen.
Note:  
If you click return before saving, the modified configuration parameters will not be saved.
Save
Click to save the modified configuration parameters if any.
