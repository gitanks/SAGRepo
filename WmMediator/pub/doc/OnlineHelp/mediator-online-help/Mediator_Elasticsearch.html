Administering Mediator : Elasticsearch Configuration
Elasticsearch Configuration
You can select Elasticsearch as a destination from CentraSite. This enables Mediator to send data for run-time events and Key Performance Indicator (KPI) metrics to the Elasticsearch instance configured in Mediator.
You can view the run-time events and metrics using third party interactive dashboards. For example, you can use Kibana as a third party interactive dashboard. You must configure the same Elasticsearch configurations in Kibana as configured in Mediator to display the run-time events and KPI metrics in the Kibana dashboard.
Click Edit to edit the configuration parameters.
The various parameters that can be configured are:
Elasticsearch Destination Configuration
Parameter
Description
Protocol
HTTP (the default) or HTTPS.
Host Name
The host name or IP address of the machine on which Elasticsearch server is running.
Port
The port that points to the Elasticsearch server. The default value is 9200.
User Name
The Elasticsearch user name that Mediator must use to access Elasticsearch.
If you are using the Operating System auth mechanism, use the following format for the user name: ES-Host-Name\ES-user-name.
Password
The Elasticsearch password that Mediator must use to access Elasticsearch.
Retype Password
The same password you entered in the Password field.
Event Types
Parameter
Description
Error
Select to publish Error events. An Error event occurs each time an invocation of a virtual service results in an error.
Lifecycle
Select to publish Lifecycle events. A Lifecyle event occurs each time Mediator starts or shuts down.
Policy Violation
Select to publish Policy Violation events. A Policy Violation event occurs each time an invocation of a virtual service violates a run-time policy that was set for the virtual service.
Performance Metrics
Parameter
Description
Publish Interval (minutes)
Specify how often (in minutes) Mediator must report performance metrics. Enter a value from 1 through 60. The default is 60.
Note:  
You can only enter a value for this parameter if Report Performance Data is selected.
