Consumer Application Details : View Consumer Application
View Consumer Application
A consumer application is an Application asset that consumes (invokes) assets (Web services, XML services or REST services) at runtime. Typically, you create consumer applications to specify the consumers that are allowed to consume a particular asset. Use the Consumer Application screen to display consumer applications configured in CentraSite.
Select and expand a consumer application to view more details about it. The various details visible are:
Parameter
Description
Description
A descriptive information of the consumer application asset. For example, consumer application for host name.
Created
The date and timestamp when the consumer application asset was created. For example, 2015-12-21 11:22:14 IST.
Modified
The date and timestamp when the consumer application asset was modified. For example, 2015-12-26 11:22:14 IST.
Tokens
The consumer application identification token. For example, User.
IP Ranges
A range of IP addresses. For example, 192.168.0.0 and 192.168.0.10
Certificate Information
The consumers based information in an X.509 v3 certificate.
The various operations performed in this screen are:
Command
Action
Expand All
Click to expand a consumer application to view more details about it.
Collapse All
Click to collapse the consumer application details.
