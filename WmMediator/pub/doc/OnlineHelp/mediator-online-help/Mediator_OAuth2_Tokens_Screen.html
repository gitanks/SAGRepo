Consumer Application Details : View OAuth2 Tokens
View OAuth2 Tokens
CentraSite supports the OAuth2 token mechanism to create a unique identifier to be used for consuming APIs.
Use the OAuth2 Tokens screen to display the list of all the consumers using the OAuth2 token for consuming APIs.
Select and expand a consumer application to view more details about it.
Select and expand a consumer application to view more details about it. The various details visible are:
Parameter
Description
ClientID
The OAuth credential used as an authorization grant to identify and authenticate client to the Integration Server.
Service Name
The name of the service that caused the event.
Expiration
The date and timestamp when the service gets expired.For example, 2015-12-21 11:22:14 IST.
The various operations performed in this screen are:
Command
Action
Expand All
Click to expand a consumer application to view more details about it.
Collapse All
Click to collapse the consumer application details.
